Split MKV

Split an MKV file into OGG files. Yay!

This tool lets you:

  • Parse an MKV file
  • Inspect the chapters
  • Enter metadata
  • Save metadata to cookie
  • Export all or some chapters to WAV or OGG

To use it:

  1. Procure an MKV file (possibly using MakeMKV)
  2. Select the MKV
  3. Enter metadata
  4. Select which chapters you want to extract
  5. Press the button 'Extract'
